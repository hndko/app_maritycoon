# PRD - Monopoli Indonesia Multiplayer Web

## 1. Overview

### Product Name

Monopoli Indonesia Online

### Product Vision

Membangun permainan Monopoli Indonesia berbasis web multiplayer real-time yang memungkinkan siapa saja membuat server permainan sendiri, membagikan link atau kode room kepada teman, dan bermain bersama tanpa perlu instalasi aplikasi.

### Target Users

- Teman bermain online
- Komunitas board game
- Pelajar dan mahasiswa
- Keluarga
- Streamer dan content creator

---

# 2. Goals

## Goals

- Multiplayer real-time hingga 8 pemain
- Host dapat membuat server/game room sendiri
- Room dapat dibagikan menggunakan:
  - URL Link
  - Room Code

- Mendukung:
  - Public Room
  - Private Room (Password)
  - Private Room (Invite Only)

- Bisa dimainkan di Desktop dan Mobile Browser
- Tidak perlu login untuk bermain

---

# 3. Core Features

## 3.1 Create Server / Room

Host dapat membuat room baru.

### Input

- Room Name
- Max Players (2-8)
- Visibility
  - Public
  - Private

- Password (optional)
- Turn Timer
- Starting Money

### Output

System menghasilkan:

- Room ID
- Room Code
- Share URL

Contoh:

https://monopoli.id/room/ABCD1234

Room Code:

ABCD1234

---

## 3.2 Join Room

### Join via Link

User membuka URL room.

### Join via Code

User memasukkan kode room.

### Validation

Jika room:

- Public → langsung masuk
- Private tanpa password → masuk
- Private dengan password → diminta password

---

## 3.3 Public Lobby

Menampilkan daftar room publik.

### Filter

- Jumlah pemain
- Sedang berlangsung
- Menunggu pemain
- Full Room

### Data

- Room Name
- Players Count
- Host Name
- Status

---

# 4. Gameplay

## 4.1 Turn System

Flow:

1. Giliran pemain
2. Roll Dice
3. Pindah token
4. Jalankan event
5. Beli properti / lelang
6. End Turn

---

## 4.2 Dice System

Dua buah dadu.

Range:

2 - 12

Rules:

- Double → bermain lagi
- Triple double → masuk penjara

---

## 4.3 Property System

Setiap kota Indonesia memiliki:

- Nama
- Harga beli
- Harga sewa
- Harga rumah
- Harga hotel

Contoh:

- Jakarta
- Bandung
- Surabaya
- Medan
- Makassar
- Yogyakarta
- Denpasar
- Balikpapan

---

## 4.4 Property Ownership

Status:

- Unowned
- Owned
- Mortgaged

Actions:

- Buy
- Sell
- Mortgage
- Unmortgage

---

## 4.5 Building System

Jika memiliki satu set warna:

Player dapat:

- Build House
- Build Hotel

Rules mengikuti Monopoli standar.

---

## 4.6 Rent System

Jika pemain berhenti pada properti lawan:

Membayar rent otomatis.

Rent tergantung:

- Properti
- Jumlah rumah
- Hotel

---

## 4.7 Chance Card

Kartu Kesempatan Indonesia.

Contoh:

- Dapat bantuan UMKM Rp500.000
- Bayar pajak kendaraan Rp300.000
- Maju ke Jakarta
- Mundur 3 langkah
- Dapat bonus THR

---

## 4.8 Community Chest

Contoh:

- Menang lomba desa
- Dapat beasiswa
- Bayar biaya rumah sakit
- Dapat warisan

---

## 4.9 Jail System

Masuk penjara jika:

- Triple double
- Kartu tertentu
- Mendarat di penjara

Keluar dengan:

- Bayar denda
- Kartu bebas penjara
- Roll double

---

## 4.10 Bankruptcy

Jika uang pemain < kewajiban.

Player dapat:

- Menjual aset
- Mortgage properti

Jika gagal:

Status:

BANKRUPT

Keluar dari permainan.

---

# 5. Multiplayer Architecture

## Room Based

Setiap room memiliki state terpisah.

Contoh:

Room A
Room B
Room C

Semua berjalan independen.

---

## Real-Time Communication

Menggunakan:

Socket.IO

Event:

- player_joined
- player_left
- game_started
- dice_rolled
- player_moved
- property_bought
- property_sold
- rent_paid
- turn_changed
- chat_message
- game_finished

---

## Reconnect System

Jika pemain disconnect:

- Data tetap tersimpan
- Slot pemain tetap ada
- Bisa reconnect

Timeout:

5 menit

---

# Sitemap & Routing

Aplikasi MariTycoon menggunakan routing berbasis path. Berikut adalah daftar halaman dan alurnya:

## 1. Public Pages

### `/` (Home)
- **Fungsi:** Halaman utama aplikasi, landing page.
- **Fitur:**
  - Tombol "Create Room"
  - Tombol "Join Room"
  - Menampilkan daftar Public Rooms yang sedang aktif
  - Panduan singkat "How To Play"
- **Akses:** Semua user (Guest).

### `/create-room` (Create Room)
- **Fungsi:** Halaman form untuk membuat room/server game baru.
- **Input:** Room Name, Max Players, Visibility (Public/Private), Password, Starting Money, dll.
- **Aksi:** Submit form akan menghasilkan Room ID dan redirect ke `/room/:roomId`.

### `/join` (Join Room)
- **Fungsi:** Halaman untuk masuk ke dalam room menggunakan Room Code.
- **Input:** Room Code (contoh: `ABCD1234`).
- **Aksi:** Validasi kode. Jika private, akan muncul prompt password. Jika berhasil, redirect ke `/room/:roomId`.

## 2. Game Pages

### `/room/:roomId` (Game Room / Waiting Room)
- **Fungsi:** Halaman utama tempat permainan berlangsung. Memiliki 2 state (Status):
  
  **State 1: Waiting Room (Sebelum Game Mulai)**
  - Menampilkan daftar pemain yang sudah bergabung.
  - Menampilkan Room Code dan Share URL untuk mengundang pemain lain.
  - Host bisa mengubah pengaturan (jika diizinkan) dan memulai game (Start Game).
  - Pemain lain bisa toggle "Ready".

  **State 2: Game Screen (Game Berlangsung)**
  - Menampilkan Papan Permainan (Board).
  - Sidebar daftar pemain (posisi, uang, status).
  - Panel Aksi (Roll Dice, Buy Property, End Turn).
  - Panel Chat & Game Logs real-time.

## 3. Alur Pengguna (User Flow)

1. **Host Flow:**
   Home `/` -> Klik "Create Room" -> Isi Form di `/create-room` -> Masuk ke `/room/:roomId` (Waiting) -> Klik "Start Game" -> Bermain.

2. **Player Flow (Via Link):**
   Klik Share URL `.../room/:roomId` -> Validasi otomatis -> Masuk ke `/room/:roomId` (Waiting) -> Menunggu Host Start -> Bermain.

3. **Player Flow (Via Code):**
   Home `/` -> Klik "Join Room" -> Masuk kode di `/join` -> Masuk ke `/room/:roomId` (Waiting) -> Bermain.

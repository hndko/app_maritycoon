# API & WebSocket Specification

Aplikasi menggunakan REST API untuk operasi di luar game (Lobby) dan WebSocket (Socket.IO) untuk in-game agar real-time.

## 1. REST API

### `POST /api/rooms`
Membuat room baru.
- **Body:** `{ room_name, max_players, is_public, password (optional), starting_money }`
- **Response:** `{ room_id, room_code }`

### `GET /api/rooms/public`
Mendapatkan daftar public room yang aktif.
- **Response:** `[ { room_id, room_name, current_players, max_players, status } ]`

### `POST /api/rooms/join`
Validasi untuk masuk room dengan code.
- **Body:** `{ room_code, password (optional) }`
- **Response:** `{ room_id, status: "success" | "need_password" | "full" }`

---

## 2. WebSocket Events (Socket.IO)

Koneksi terjadi di namespace atau menggunakan room feature di Socket.IO.

### Client to Server (Emit dari Frontend)

- **`join_room`**
  - Payload: `{ room_id, user_nickname }`
  - Aksi: Memasukkan socket user ke room tersebut.

- **`chat_message`**
  - Payload: `{ message }`
  - Aksi: Mengirim pesan ke semua orang di room.

- **`start_game`** (Only Host)
  - Payload: `{}`
  - Aksi: Memulai permainan, mengacak giliran, set status game ke 'playing'.

- **`roll_dice`**
  - Payload: `{}`
  - Aksi: Server mengacak dadu (2-12), menghitung pergerakan pemain.

- **`buy_property`**
  - Payload: `{ property_id }`
  - Aksi: Player membeli properti tempat dia mendarat.

- **`build_house`**
  - Payload: `{ property_id }`
  - Aksi: Membeli rumah/hotel di properti yang dimiliki (jika punya 1 set warna).

- **`end_turn`**
  - Payload: `{}`
  - Aksi: Mengakhiri giliran, server memindahkan giliran ke pemain berikutnya.

- **`pay_rent`**
  - Payload: `{}`
  - Aksi: Membayar sewa ke pemilik properti.

### Server to Client (Listen di Frontend)

- **`room_state_update`**
  - Payload: `{ players: [], properties: [], current_turn: "uuid" }`
  - Deskripsi: Update state global setiap ada perubahan besar (orang masuk/keluar, atau saat connect).

- **`game_started`**
  - Payload: `{ first_turn_player_id }`

- **`chat_broadcast`**
  - Payload: `{ sender_name, message, type: "user"|"system", timestamp }`

- **`dice_rolled_result`**
  - Payload: `{ player_id, dice_1, dice_2, total, is_double }`

- **`player_moved`**
  - Payload: `{ player_id, new_position }`

- **`action_required`**
  - Payload: `{ type: "buy_property" | "pay_rent" | "chance_card", data: {} }`
  - Deskripsi: Server meminta pemain yang sedang giliran untuk melakukan aksi tertentu.

- **`property_updated`**
  - Payload: `{ property_id, owner_id, house_count, hotel_count }`

- **`turn_changed`**
  - Payload: `{ next_player_id }`

- **`player_bankrupt`**
  - Payload: `{ player_id }`

- **`game_finished`**
  - Payload: `{ winner_id, leaderboard: [] }`

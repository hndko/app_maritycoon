# Components Architecture

Daftar komponen UI yang digunakan dalam aplikasi MariTycoon (berbasis React/Next.js).

## 1. Global / UI Core Components
Komponen dasar yang digunakan di seluruh aplikasi (Design System).

- **`Button`**: Tombol aksi utama (Primary, Secondary, Danger, Success).
- **`Input`**: Form input untuk text, number, dan password.
- **`Modal` / `Dialog`**: Overlay untuk konfirmasi, form, atau informasi detail (cth: Trade Dialog, Winner Dialog).
- **`Toast` / `Snackbar`**: Notifikasi sekilas (cth: "Berhasil copy link", "Gagal join room").
- **`Avatar`**: Menampilkan ikon/gambar profil pemain sementara.
- **`Badge`**: Label status (cth: "Host", "Ready", "Bankrupt").
- **`Card`**: Container dasar untuk membungkus konten (cth: Room Card di public lobby).
- **`Tooltip`**: Informasi tambahan saat hover di atas elemen tertentu.
- **`LoadingSpinner`**: Indikator proses loading.

## 2. Page Specific Components

### Home & Lobby
- **`RoomList`**: Menampilkan grid/list dari `RoomCard`.
- **`RoomCard`**: Komponen yang menampilkan nama room, jumlah pemain, dan tombol join.

### Game Screen: Papan & Properti
- **`GameBoard`**: Komponen utama pembungkus seluruh papan Monopoli.
- **`BoardTile`**: Kotak individual di papan (bisa berupa Kota, Penjara, Start, Chance, dll).
- **`PropertyCard`**: Desain kartu sertifikat tanah (menampilkan harga beli, sewa, dan biaya rumah/hotel).
- **`PlayerToken`**: Pion pemain yang bergerak di atas `BoardTile`.
- **`Dice`**: Komponen animasi dadu 3D/2D yang menunjukkan angka lemparan.

### Game Screen: Interaksi
- **`PlayerSidebar` / `PlayerList`**: Daftar pemain beserta status uang dan urutan giliran.
- **`PlayerCard`**: Detail satu pemain di sidebar.
- **`TurnIndicator`**: Menandakan giliran siapa saat ini (animasi highlight).
- **`ActionPanel`**: Kumpulan tombol aksi (Roll Dice, End Turn, Pay Rent, Build House).
- **`ChatBox`**: Komponen untuk chat antar pemain di room yang sama.
- **`GameLog`**: Menampilkan history event (cth: "Budi melempar dadu 8", "Siti membeli Jakarta").

## 3. Modals & Dialogs (In-Game)
- **`PropertyDetailModal`**: Muncul saat pemain klik sebuah kotak/properti untuk melihat detail harga.
- **`PurchaseModal`**: Prompt apakah pemain ingin membeli properti saat berhenti di kotak kosong.
- **`TradeDialog`**: Interface untuk bertukar properti/uang dengan pemain lain.
- **`BankruptcyModal`**: Peringatan saat uang tidak cukup, meminta pemain menjual/mortgage aset.
- **`WinnerDialog`**: Tampilan akhir game merayakan pemenang dengan efek confetti.

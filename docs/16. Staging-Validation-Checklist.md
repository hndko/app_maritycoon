## 10. Production Gate

Production deployment boleh dilanjutkan hanya jika:

- Semua item 2 pemain `PASS`.
- Semua item 4 pemain `PASS`.
- Semua item 8 pemain critical flow `PASS`: create room, join room, start game, reconnect, disconnect, turn timer, property buy, rent, winner.
- Tidak ada fatal browser console error.
- Tidak ada backend unhandled exception.
- `/api/health` tetap `ok`.
- Backup sebelum test tersedia.
- Restore minimal pernah diverifikasi di staging dalam siklus release yang sama.
